# FUTURE_ML_02
🚀 Support Ticket Classification & Prioritization System
📌 Project Overview

This project builds a Machine Learning model that automatically classifies customer support tickets into categories such as:

Account

Billing

Technical Issue

General Query

It also predicts the priority level (High, Medium, Low).

🛠 Technologies Used

Python

Pandas

Scikit-learn

TF-IDF Vectorization

Naive Bayes Classifier

📊 Model Performance

Category Classification Accuracy: 100%

Priority Prediction Accuracy: 93.33%

🔍 Features

Text preprocessing using TF-IDF

Automatic ticket categorization

Priority prediction

Confusion Matrix & Classification Report

Custom ticket testing
